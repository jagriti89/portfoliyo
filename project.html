URL-Shortner

[GitHubLink]

Role: Backend Developer,
Tech Stack: Javascript, Node, MongoDB.
URL shortening is used to create shorter aliases for long URLs. We call these we use Redis, shortened aliases “short links.” Users are redirected to the original URL when they hit these short links.
Short links save a lot of space when displayed, printed, messaged, or tweeted. Additionally, users are less likely to mistype shorter URLs.
